# Winux
the Automaited installing Winux Using Arch and Winux Runtime installer
# Installing 
Boot your Arch from Iso 
type this command 
and wait
```
curl -O https://raw.githubusercontent.com/AGlegendery/Winux/refs/heads/main/Winux-Installer
```
and
```
chmod +x ./Winux-Installer
./Winux-Installer
```
congrats your winux is ready
